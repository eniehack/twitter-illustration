# twitter-illustration

Twitter をイラスト共有サイトのように使用するためのアプリケーション

## 環境構築

### フロントエンド

```bash
npm run start  # localhost:3000 で起動
```

### バックエンド

初回仕様時は DB のマイグレーションを行う。

```bash
npx sequelize-cli-esm db:migrate
```

環境変数 `.env` を設定する。


起動

```bash
cd backend
npm run start
```
